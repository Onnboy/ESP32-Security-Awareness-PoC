# ESP32-Security-Awareness-PoC
Estudo de Prova de Conceito (PoC) focado em Conscientização de Segurança Cibernética. Implementação de um Captive Portal educativo em ESP32 que demonstra os riscos de redes Wi-Fi públicas e ataques de Engenharia Social, alertando o usuário em tempo real sobre a exposição de dados!
